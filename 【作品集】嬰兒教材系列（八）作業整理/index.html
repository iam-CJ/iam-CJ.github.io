【作品集】嬰兒教材系列（八）作業整理

發表於 2023-01-01 分類於 作品集

說明

為方便於履歷呈現我過往的練習，新增一篇文章將這系列作業整理於此

作業

第一課－碼表應用程式

作業說明：做出簡易的計時碼表，需要有開始按鈕、重設按鈕，按下開始按鈕後碼表開始計時，並且開始的字樣變成暫停，再按下暫停按鈕，字樣變成繼續，而按下重設按鈕碼表要歸零。
https://jsfiddle.net/iamCJ/2hLfa0tw/

第二課－筆記應用程式

作業說明：做出簡易的記事本，有一個可以輸入文字的textarea還有新增筆記的按鈕，筆記內容要隨時存到 local storage，還需要有個筆記清單的區塊來放置所有事項，每個事項旁都有刪除按鈕，且點擊單一事項可以修改該事項的內容
https://jsfiddle.net/iamCJ/21j9qdv8/

第三課－分帳應用程式

作業說明：做出簡易的分帳程式
作業要求如下：（1）資料要隨時存入 local storage（2）需要有可以增加成員姓名、增加款項的按鈕，而增加款項的部分可以填入金額、事項以及由誰付款（3）需要成員列表及款項列表將所有成員及所有款項列出，成員列表的部分需列出成員姓名以及應付/應收金額（並需有顏色區別），款項列表則需列出金額、事項以及付款人，並且有刪除按鈕可以刪除個別款項（4）需要有旅程結束的按鈕，清除所有成員及款項資料
https://jsfiddle.net/iamCJ/xgu5k1vy/

第四課－行事曆應用程式

作業說明：做出簡易的行事曆，需要有一個新增活動的按鈕，點擊按鈕後可以輸入活動名稱、選擇活動日期與時間，選擇日期時間的 UI 用之前課程教過的 datetimepicker 套件，而行事曆 UI 的部分則使用 FullCalendar 套件，需要將活動正確顯示於行事曆上，並且點擊活動可以出現確認刪除的對話方塊，確認後能刪除活動，並且使用 CDN 版本，自己學習如何在框架內導入套件，不能用官方提供的 Vue 版本
https://jsfiddle.net/iamCJ/cxrqgt0j/

第五課－體重追蹤應用程式

作業說明：做出簡易的體重追蹤工具，需要有可以輸入體重及日期的表格，以及新增資料的按鈕，並且有清單列表將所有資料依日期排序列出，搭配 Chart.js 將資料圖表化（以折線圖的形式呈現）
https://jsfiddle.net/iamCJ/tncuzgd8/

第六課－天氣查詢應用程式

作業說明：做出簡易的天氣查詢工具，需要有選單選取縣市，選取縣市後需要呈現該地的天氣現象、最高溫度、最低溫度、降雨機率，而資料取用自中央氣象局的 API
https://jsfiddle.net/iamCJ/9odgx75c/

第七課－國家資訊與地圖應用程式

提醒：本次作業用到的 restcountries API，不時可能因點擊數太多、伺服器負荷量爆炸，抑或作者暫時關閉 API，導致呼叫 API 失敗而無法正常顯示

作業說明：做出簡易的國家資訊與地圖工具，顯示世界地圖，在每個國家的中心點放置大頭針的標記，點擊大頭針後會跳出視窗，視窗內顯示國旗圖案、國家名稱、首都名稱、貨幣符號
https://jsfiddle.net/iamCJ/m30ayj8g/
