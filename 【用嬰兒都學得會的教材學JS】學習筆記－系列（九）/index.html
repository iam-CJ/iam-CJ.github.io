【用嬰兒都學得會的教材學JS】學習筆記－系列（九）

發表於 2023-05-29 分類於 JavaScript

前言小廢話

這系列主要是學習 Vue 元件的觀念，不過由於站長大大在課內並無多敘述、教學，大都是要我們去讀官方文件，再加上我在之前就透過 Alex 大大的影片學習過相關概念、也做了筆記，所以建議想要更深入了解 Vue 元件的觀念可以參考 這邊 ，也因此我在這篇的心得及技術就沒有多著墨了，造成不便請見諒 ∠( ᐛ 」∠)＿

課程開始

第一課－學習 Vue 元件基本觀念

  • 課程概要：認識 Vue 元件的基本觀念、註冊方法以及元件的檔案格式
  • 作業說明：建立三個元件，各元件需顯示指定內容，並將三個元件匯入App.vue

    https://reurl.cc/M8ZjAX

第二課－學習 Vue 的 props 觀念

  • 課程概要：認識 Vue props 觀念
  • 作業說明：仿製 Quasar framework 中的 Badge 徽章元件，建立MyBadge.vue元件，並能夠透過屬性設定背景顏色、文字顏色、文字內容以及 rounded 樣式，最後匯入App.vue

    https://reurl.cc/aV58oX

第三課－學習 Vue 的 events 觀念

  • 課程概要：學習在 component 內寫事件、認識$emit
  • 作業說明：仿製 Quasar framework 中的 Button 按鈕元件，建立MyButton.vue元件，並能夠透過屬性設定背景顏色、文字顏色、文字內容以及 rounded 和 square 樣式，還有透過$emit設定點擊按鈕後的事件，最後匯入App.vue

    https://reurl.cc/OvXnlg

第四課－學習 Vue 的 v-model 觀念

  • 課程概要：認識v-model
  • 作業說明：仿製 Quasar framework 中的 Input 元件，建立MyInput.vue元件，並能夠透過屬性設定欄位名稱、 filled 和 outlined 樣式，還有透過v-model設定值、取得值，最後匯入App.vue

    https://reurl.cc/4oyOlY

第五課－學習 Vue 的 slots 觀念

  • 課程概要：認識 Vue slots 觀念
  • 自己補充：子元件可以有多個 slot，在 slot 中加入name屬性並取名，在父元件加入template tag 並使用v-slot:slotname或縮寫成#slotname就可以分別指定顯示內容，這就是具名插槽，下方是實際用法
  • 作業說明：仿製 Quasar framework 中的 Banner 公告元件，建立MyBanner.vue元件，能夠透過屬性設定 rounded 樣式，以及使用slot設定顯示內容，最後匯入App.vue

    https://reurl.cc/qLmlQN

第六課－學習用外部狀態管理 vue 元件

  • 課程概要：學習用外部狀態管理 vue 元件
  • 作業說明：仿製 Quasar framework 中的 Dialog 視窗元件，建立MyDialog.vue元件，能夠透過v-model管理 dialog 元件是否顯示，以及使用slot設定顯示內容，最後匯入App.vue

    https://reurl.cc/QX70eZ

第七課－學習開發狀態複雜的 vue 元件

  • 課程概要：學習開發狀態複雜的 vue 元件
  • 關鍵字：hex to rgb javascript、rgb to hex javascript
  • 作業說明：仿製 Quasar framework 中的 Color Picker 元件，建立MyColor.vue元件，使用v-model設定 HEX 格式色碼，並有可以拖動的捲軸選擇 RGB 比例，最後匯入App.vue

    https://reurl.cc/WG31dO

小結

終於！把所有系列完結了！趕快改履歷找工作囉！XD
不得不說上完整系列的課，讓我更加熟悉 JavaScript、框架的運用，除了了解到自己不足的地方，其實也對自己能夠獨立完成這麼多作業感到驚訝，過去我總覺得自己學得不夠、比不過其他人，所以就一直埋頭苦學、學到很茫然，直到學完這些課後，真的對自己有了一咪咪的自信心了，我要勇敢跨出第一步，邁向成為前端工程師的目標！

文章連結
