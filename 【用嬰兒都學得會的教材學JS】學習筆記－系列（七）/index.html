第三課－學習 Vue 條件判斷式的寫法

  • 課程概要：
    1. 認識綁定 class 以及 style 的寫法：首先 Vue 有屬性綁定資料的功能，語法是v-bind:attribute，可以縮寫為:attribute
      綁定 class 以及 style 實在有很多寫法，這邊只寫基本用法，如果有需要可以直接去參考官方文件
      1. 綁定 class 的方法：請參照下方範例，在 template 的 tag 內加入:class，而大括弧內前面放 class 的名字，後面放型態為布林值的參數，可以用於根據需求切換 class
      2. 綁定 style 的方法：請參照下方範例，在 template 的 tag 內加入:style，而大括弧內前面放屬性，後面放參數，如果屬性名稱以-連接，可以用引號包起來，或者用駝峰式寫法
    2. 認識條件判斷的寫法：依照需求，在 template 的 tag 內加上v-if、v-else、v-else-if或v-show，用法基本上跟在 JS 內一樣，可以放參數或是判斷式，當判斷式的結果為true時，就會在 template 出現該 tag 內容，其他不符條件的就不會出現
    3. 補充：
      1. v-if和v-show的差別：這兩者看起來的效果相同，不過v-if控制的是 DOM 元素，不符條件的會被 Vue 把內容抽起來並註解掉那段 tag，而v-show則是控制 CSS display 屬性，不符條件的會被設為display:none
        因此v-if會花費較多的切換成本，而v-show會花費較多的載入成本（因為要將全部條件都載入網頁），所以如果是要用在頻繁切換的功能，建議使用v-show
      2. 官方文件不建議在同一層級同時使用v-if和v-for，牽涉到兩者優先度的設定，v-if會先被執行
  • 作業心得：作業不難，難的是寫文章_(´ཀ`」 ∠)_

    作業說明：延續上一課作業，將 data 的 color 綁定到 template 的 style

第四課－學習 Vue 迴圈的寫法

  • 課程概要：認識 for 迴圈寫法v-for，像是結合for of、for in，可以同時取到陣列的值及索引，也可以只用一個參數取值，中間連接詞寫in或of皆可，功能是一樣的
  • 作業心得：這個也是之前常常寫到的，輕輕鬆鬆啦(`･∀･)b

    作業說明：延續上一課作業，將原先一筆一筆 render 的事項改用 for 迴圈寫

第五課－學習 Vue 事件處理的寫法

  • 課程概要：
    1. 認識事件處理的寫法v-on:eventname：可以縮寫為@eventname，後面可以放函式名稱，也可以直接執行函式
    2. 如何分辨 template 中雙引號的內容，是變數還是字串：
      基本上在屬性前面帶有 Vue 的語法（如：v-系列、@、:等），雙引號內容會是變數，其他時候就跟 HTML 的屬性一樣，裡面放的是字串
    3. 認識處理表單資料的方法：可以使用v-model將使用者在表單輸入的值做資料雙向綁定，能夠在輸入的值改變時，資料同步更新在 template 和 data 中
      不過在 Vue 3，v-model的用法跟寫法修改很多，建議在轉換使用 Vue 3 時，可以多看v-model的說明
  • 作業心得：由於太不習慣 Vue 1 的寫法，我改成用 Vue 2 了 ∠( ᐛ 」∠)＿5/31 更正：站長大大給的文件版本是 Vue 3

    作業說明：接續上一課作業，增加新增事項與刪除事項的功能，會運用到v-model、綁定事件的方法
    https://jsfiddle.net/iamCJ/1tawvj6r/

第六課－認識 Lifecycle Hooks 與 Watchers

  • 課程概要：
    1. 認識 Lifecycle Hooks：首先要先認識 Vue 的生命週期，Vue 的實體物件（元件 component），有被建立、更新、銷毀的生命週期階段，而每個階段皆有 Hook 讓我們可以在該階段執行一些行為或檢查
      p.s.而更詳細的部分我之前有做過筆記，待之後補上
    2. 認識 Watcher 屬性：根據我在 Alex 大大的影片學到的，watch主要管理資料產行為/事件，用於隨時監聽某些變數，如果發現它們更新，就執行我們設定的行為或事件，而這樣的行為或事件被稱為副作用(side effects)
    3. 補充：遇到陣列或物件型態的 data，需要深度觀察、將deep屬性設為 true，watcher 才會監聽到物件內的所有屬性，但原理不太清楚（？
  • 作業心得：感覺我做作業做到上癮（？，用已經熟練的框架寫程式寫得好快樂 XD，加上之前就有做過相關的筆記，這課的筆記算是寫得滿輕鬆的

    作業說明：接續上一課作業，增加自動儲存的功能，使用 watcher 監聽 data 的notes，一有更新資料就將notes存入 local storage，並且在mounted階段檢查 local storage 有無notes的資料，如果有則自動載入
    https://jsfiddle.net/iamCJ/d36hL2r5/

第七課－認識 Template Refs

  • 課程概要：
    1. 認識什麼是 Template Refs：框架會幫我們處理及更新 DOM 相關的工作，若我們再自行去操作 DOM 可能會發生難以處理的錯誤，因此 Vue 提供了ref這個屬性來解決抓取 DOM 的問題
      在想要抓取的 template tag 內加入ref=""，雙引號內填入 ref value（可以理解成把這個 reference 命名），在 JS 只要使用this.$refs.refValue就可以抓取到這個 DOM 元素
  • 作業心得：今天學到新的東西！在這之前我抓取 DOM 元素都是用querySelector，不過好像也沒有遇到過什麼問題 ∠( ᐛ 」∠)＿

    作業說明：使用ref抓取textarea，在textarea套用之前課程介紹過的 autosize 套件
    https://jsfiddle.net/iamCJ/5tpobqdn/
